Basic Process

How the Homepage Supports the IMS

The homepage provides an entry point for monitoring and accessing the main record modules.

1

Open IMS Homepage

The user opens the web-based IMS homepage and views system overview, modules, and analytics.

2

Select Module

The user selects a record module such as communication, travel order, memo, inventory, or visitor log.

3

Monitor Records

The dashboard charts help users view document flow, pending transactions, and archived records.

4

Access Dashboard or Logout

The profile menu allows the user to proceed to the dashboard, account settings, or logout.

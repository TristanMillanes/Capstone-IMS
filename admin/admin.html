Administrative Tasks Overview

Follow these key steps to effectively manage the IMS system and users.

1

Access Admin Dashboard

Login to the admin panel and view system overview, active users, and pending tasks.

2

Review Account Requests

Check and approve or deny pending user account requests from the account management module.

3

Monitor System Activity

Review audit logs and activity reports to track user actions and system performance.

4

Maintain System Security

Configure security settings, backup data, and manage user permissions as needed.
